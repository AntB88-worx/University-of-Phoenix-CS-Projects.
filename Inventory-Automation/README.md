# 📦 Inventory Automation & Logistics Optimization Hub

This folder serves as an independent portfolio showcasing how I leverage scripting, data analytics, and cloud logic to streamline enterprise supply chains, automate warehouse workflows, and eliminate operational bottlenecks.

---

### 🚀 Core Business & Technical Objectives
* **Data-Driven Efficiency:** Replacing manual Excel tracking with localized Python scripts for automated data extraction.
* **Proactive Logic:** Writing predictive scripts to alert teams when physical stock levels drop below dynamic thresholds.
* **Process Scaling:** Conceptualizing cloud-based data microservices to manage multi-node inventory ecosystems.

---

### 🛠️ Portfolio Projects

#### 🐍 1. Python Inventory Control Script (`inventory_manager.py`)
* **The Problem:** Manual cycle counts are time-consuming and prone to human data-entry errors.
* **The Solution:** A script that parses raw warehouse CSV payloads, checks quantities against minimum baseline thresholds, and flags variances instantly.
* **Tech Used:** `Python 3.x`, `CSV modules`, `File I/O operations`
* **Status:** ⏳ *In Development*

#### ☕ 2. Java Order Fulfillment Tracker (`FulfillmentTracker.java`)
* **The Problem:** Tracking order statuses across multiple staging steps (Picked, Packed, Shipped) without a centralized application.
* **The Solution:** An object-oriented program mapping operational workflows to strict data objects to maintain integrity throughout fulfillment lifecycles.
* **Tech Used:** `Java 17+`, `OOP paradigms (Inheritance, Encapsulation)`
* **Status:** ⏳ *In Development*
