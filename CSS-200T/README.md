# 🐍 CSS-200T: Foundations of Computer Science

Welcome to my academic directory for the CSS-200T course at the University of Phoenix. This folder contains foundational programming projects designed to master computational thinking, core data structures, and algorithmic logic.

---

### 🧠 Core Learning Objectives
* **Control Flow:** Implementing complex conditional branching (`if/elif/else`) and loops (`for/while`) to handle dynamic inputs.
* **Functional Programming:** Modularizing code into clean, reusable functions and static methods.
* **Data Typings:** Managing numerical data constraints, type casting, and string formatting across multiple languages.

---

### 📂 Included Course Projects

#### 📈 1. Student Performance Tracker (`grade_calculator.py`)
* **Concepts Demonstrated:** Arrays/Lists, arithmetic summation, automated text filtering, and float formatting.
* **Functionality:** Evaluates a series of academic scores, calculates the final average, assigns a letter grade, and automatically flags underperforming elements for review.

#### 🌡️ 2. Computational Metric Converter (`TempConverter.java`)
* **Concepts Demonstrated:** Static methods, array iteration, data-type consistency (`double`), and tabular console spacing.
* **Functionality:** Takes an array of raw Fahrenheit benchmarks and applies functional equations to map them into Celsius equivalents in a structured table.

---

### 💻 Setup and Execution
To run these files locally on your machine, navigate to this directory and execute the following commands via your terminal:

```bash
# To run the Python Script
python grade_calculator.py

# To compile and run the Java Script
javac TempConverter.java
java TempConverter
```
