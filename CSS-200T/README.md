# 🐍 CSS-200T: Foundations of Computer Science

Welcome to my academic directory for the CSS-200T course at the University of Phoenix. This folder contains foundational programming projects designed to master computational thinking, core data structures, and algorithmic logic.

---

### 🧠 Core Learning Objectives
* **Control Flow:** Implementing complex conditional branching (`if/elif/else`), infinite loops (`for/while`), and cross-platform terminal clear-screen states.
* **Functional Programming:** Modularizing code into clean, reusable functions, static methods, and automated business alert triggers.
* **Data Typings:** Managing numerical data constraints, type casting, chronological deltas, and real-time screen rewrite wrappers (`\r`).

---

### 📂 Included Course Projects

#### 🧑‍🏭 1. Warehouse Shift Labor Productivity Tracker (`labor_productivity.py`)
* **Concepts Demonstrated:** Arrays/Lists, arithmetic summation, automated text filtering, and float formatting.
* **Functionality:** Evaluates a series of real-time employee picker metrics, calculates shift throughput averages, assigns an operational SLA efficiency tier, and flags underperforming picking stations for manager review.

#### 🌡️ 2. Cold Chain Thermal Compliance Audit System (`ColdChainCompliance.java`)
* **Concepts Demonstrated:** Static methods, array iteration, data-type consistency (`double`), and tabular console spacing.
* **Functionality:** Ingests an array of raw Fahrenheit telemetry logs from refrigerated transit trucks and applies functional conversion formulas to map them into Celsius equivalents alongside automated FDA risk assessment flags.

#### 🔢 3. EDI Manifest Data Payload Optimizer (`edi_manifest_optimizer.py`)
* **Concepts Demonstrated:** Data scale abstraction, type casting, mathematical optimization, and user-input handling.
* **Functionality:** Dynamically processes raw transaction data package bytes and translates them into bits, KB, and MB scales to verify if legacy mainframe routing systems can safely accept transmission manifests without timeouts.

#### 📝 4. Academic Milestones (`5-Week-Logic.txt`)
* **Content:** Documented structural logic and progression notes matching the course curriculum milestones.

### 💻 Setup and Execution

To run the Python automation scripts locally, execute the following commands via your terminal:

```python
python labor_productivity.py
python edi_manifest_optimizer.py
```

To compile and execute the Java tracking components, use these commands:

```java
javac ColdChainCompliance.java
java ColdChainCompliance
```
