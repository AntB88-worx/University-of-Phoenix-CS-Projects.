# Documentation: Data Converter Script (data_converter.py) 🧠

This document explains the functionality and concepts behind the bits and bytes conversion tool.

## 📂 About the Script
* **data_converter.py** - A custom tool built to experiment with data size metrics. It tracks bytes and dynamically breaks them down into bits, Kilobytes, and Megabytes to demonstrate how computer hardware structures information.

## ⚙️ Concepts Explored
- Digital data measurement units (Bits vs. Bytes)
- Binary scale data storage scaling (1,024-byte boundaries)
- Python user input handling and data sanitization
