# 🐍 CSS-200T: Foundations of Computer Science

Welcome to my academic directory for the CSS-200T course at the University of Phoenix. This folder contains foundational programming projects designed to master computational thinking, core data structures, and algorithmic logic.

---

### 🧠 Core Learning Objectives
* **Control Flow:** Implementing complex conditional branching (`if/elif/else`), infinite loops (`for/while`), and cross-platform terminal clear-screen states.
* **Functional Programming:** Modularizing code into clean, reusable functions, static methods, and automated business alert triggers.
* **Data Typings:** Managing numerical data constraints, type casting, chronological deltas, and real-time screen rewrite wrappers (`\r`).

---

### 📂 Included Course Projects

#### 📈 1. Student Performance Tracker (`grade_calculator.py`)
* **Concepts Demonstrated:** Arrays/Lists, arithmetic summation, automated text filtering, and float formatting.
* **Functionality:** Evaluates a series of academic scores, calculates the final average, assigns a letter grade, and automatically flags underperforming elements for review.

#### 🌡️ 2. Computational Metric Converter (`TempConverter.java`)
* **Concepts Demonstrated:** Static methods, array iteration, data-type consistency (`double`), and tabular console spacing.
* **Functionality:** Takes an array of raw Fahrenheit benchmarks and applies functional equations to map them into Celsius equivalents in a structured table.

#### 🔢 3. Bits & Bytes Storage Converter (`data_converter.py`)
* **Concepts Demonstrated:** Data scale abstraction, type casting, mathematical optimization, and user-input handling.
* **Functionality:** Dynamically processes raw byte data and translates it into bits, KB, and MB to visualize lower-level hardware storage frameworks.

#### 🏭 4. Supply Chain Control Tower Engine (`clock.py`)
* **Concepts Demonstrated:** Multi-hub UTC offset mappings, continuous event loop refreshing, and runtime variable intercepting.
* **Functionality:** Simulates an operational logistics dashboard tracking freight movement across DFW, LAX, and JFK hubs, monitoring real-time dispatch SLA windows and allowing shift simulation stress-testing.

#### ⏳ 5. Operational Milestone Analyzer (`countdown.py`)
* **Concepts Demonstrated:** Chronological calculations, error parsing boundaries, and single-line console carriage overwriting.
* **Functionality:** Captures dynamic distribution target deadlines, applies essential operational handling buffers, and projects compounding financial liability exposure (\$/hour) if an SLA breach occurs.

#### 📅 6. Enterprise Capacity Planner (`calendar_view.py`)
* **Concepts Demonstrated:** String matrix parsing blocks, conditional regex substitution, and calendar formatting logic.
* **Functionality:** Outputs a full 12-month calendar layout tailored for supply chain forecasting, automatically injecting Q4 Holiday peak capacity warning banners and carrier shipping holiday flags.

#### 📝 7. Academic Milestones (`5-Week-Logic.txt`)
* **Content:** Documented structural logic and progression notes matching the course curriculum milestones.

---

### 💻 Setup and Execution

To run the Python automation scripts locally, execute the following commands via your terminal:

```python
python grade_calculator.py
python data_converter.py
python clock.py
python countdown.py
python calendar_view.py
```

To compile and execute the Java tracking components, use these commands:

```java
javac TempConverter.java
java TempConverter
```
